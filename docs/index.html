Overview

What Vite Forge does

Creates a Vite app

It scaffolds a new Vite project, then layers in the tooling and files the starter expects.

Sets up the stack

React, React DOM, Tailwind CSS, PostCSS, React Router, and ESLint are installed and configured.

Writes opinionated files

The generated app already includes a router-backed structure with pages, components, hooks, and styles folders.

Quick Start

Start a new project in one command

npm install -g vite-forge
npx vite-forge my-app
cd my-app
npm run dev

Commands

CLI reference

  • npx vite-forge <project-name> - create a new project
  • --ts - use TypeScript, which is the default
  • --js - use JavaScript instead of TypeScript
  • --template basic|dashboard|landing - choose the starter template
  • --help - show the built-in help text
  • --version or -v - print the package version

Behavior

How the generator works

Non-interactive

The CLI avoids Vite prompts and automatically chooses the starter template and install flow.

Opinionated config

Tailwind, PostCSS, and ESLint are written directly so the generated app is immediately usable.

Router ready

The generated project uses React Router from the start, so multi-page apps can grow naturally.

FAQ

Common questions

Can I use JavaScript instead of TypeScript?

Yes. Pass --js when you run the CLI.

Can I change the generated code later?

Absolutely. Vite Forge is a starter, not a framework lock-in.

How do I preview this docs site locally?

Run npm run docs and open the local URL printed in the terminal.
