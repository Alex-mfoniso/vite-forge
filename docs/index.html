Introduction

Ship your React projects without the boilerplate. Vite Forge creates a ready-to-run Vite app with standard tooling configured out of the box.

Get Started GitHub

Quick Start

Start a new project in one command using your terminal.

npm install -g vite-forge
npx vite-forge my-app

Navigate into your new project and start the development server:

cd my-app
npm run dev

Commands

Vite Forge comes with a minimal CLI footprint to get you moving quickly.

--ts

Scaffolds configuration with strict TypeScript (Default behavior).

--js

Overrides standard generation pipelines to process native JavaScript.
